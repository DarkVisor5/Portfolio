To-Do List App

The "To-Do List App" is a web application designed to help users efficiently manage their daily tasks. It provides a simple and intuitive interface for adding, deleting, and marking tasks as completed. The application features a minimalist design for easy navigation and task management, enhancing user productivity with basic functionalities and advanced features like item reordering and completion marking.

Technologies used: HTML5, CSS, JavaScript, jQuery, jQuery UI.

Pokédex App

The "Pokédex App" is a JavaScript web application that allows users to browse a database of Pokémon. It loads data from the external Pokémon API and displays details about each Pokémon when selected. The goal is to demonstrate how to build a functional and aesthetically pleasing web app using modern web technologies.

Technologies used: HTML, CSS, JavaScript, external APIs (Pokémon API).

myFlix API

The "myFlix API" provides a server-side for a movies web application, where users can access detailed information about movies, directors, and genres. It allows users to register, update their profile, and manage their favorite movies list. This project demonstrates the ability to build full-stack applications using the MERN stack.

Technologies used: Node.js, Express, MongoDB, Mongoose, REST API.

myFlix React App

The "myFlix React App" is the client-side development of the myFlix project, providing a dynamic and interactive user interface for accessing movie information. Built with React, this single-page application enhances user experience by allowing responsive interactions and real-time updates without page reloads.

Technologies used: React, React Router, Redux, Bootstrap.

Meet App

The "Meet App" is a serverless, progressive web application (PWA) built using React and developed with a test-driven development (TDD) approach. It integrates with the Google Calendar API to fetch upcoming events, enhancing user experience by allowing the app to function offline and support interactive data visualizations like charts.

Technologies used: React, Google Calendar API, AWS Lambda, Serverless, TDD, PWA features.

See live version | GitHub Repository
